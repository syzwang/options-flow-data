Delta-25 Put/Call IV 偏度（按到期分桶）

偏度 = Put IV − Call IV（delta-25 行权价）。正 = put 贵（正常对冲需求）；负 = call 贵（投机 FOMO）。
安全区（偏度 > 0）— put 贵于 call，市场定价下行风险 = 健康 危险区（偏度 < 0）— call 贵于 put，投资者追涨 = 顶部信号 零轴（临界线） 现价（右轴）
ℹ️混合 — 2/5 个到期桶为负（最低 -2.4）。Call 偏仅在近端；远端仍在安全区。
✓偏度在正常区间 — 71th %ile of 14天历史，当前 -2.4 vs 中位数 -4.8）。无极端仓位信号。 · ⚠ 仅 14 天历史 — 分位数在 n≥60 前不可靠

GEX 墙 vs 成交量墙（点击）
成交量墙（下方持续放量行权价）显示散户/机构流向集中点 — 后视镜信号。
GEX 墙（本卡）显示做市商对冲压力最大处 — 前瞻性，驱动真实盘中价格行为。

成交量墙不一定是 GEX 墙（例如远期 OI gamma 低）。盘中真正起支撑/阻力作用的是 GEX 墙，因为做市商必须交易股票来对冲。两者一致 = 墙稳固；不一致 = 信 GEX。

持仓分布（按到期日）

单一到期日的逐行权价 call/put 持仓。 绿色=call，红色=put。 最大痛点（黄）= 多数期权到期作废的行权价，历史上做市商常 pin 此价位。 现价（青）= 当前股价。 现价上方真空带= 期权做市商在此区间无明显抵抗，股价容易漂移至下一组 call 堆积处。

如何按深度读 OI Delta（点击展开）
OI Delta = 两次 snapshot 之间净新建合约（买卖双方都开仓）。表面总数有误导性 — OI 落在哪里比总量更重要。

OTM Call (above spot) = directional 上行押注. Retail FOMO or institutional call buying.
ITM Call（现价之下） = 通常是 covered-call 写卖方对已持股锁利。OI 增加但并非看涨 — 是获利了结。
OTM Put（现价之下） = 下行押注或组合对冲。
Deep OTM Put (<-15% spot) = systematic 尾部保险. Only bought by institutions/macro funds. When this surges, smart money is nervous.

经典模式：
• 纯多头：OTM calls 占主导，puts 安静 → 纯上行定投
• 纯空头：OTM puts 占主导，calls 安静 → 纯下行定投
• 「系安全带」：ITM calls + 深度 OTM puts 同时膨胀 → 机构获利 + 对冲 = 晚期牛市警告
• 轮动：ITM calls 平仓（OI 下降）+ OTM calls 开仓（OI 上升）→ 多头仓位在涨后向上 roll

墙如何识别（点击展开）
Call Wall — OTM call strike (3–15% above spot) with highest persistence × avg daily volume. Dealers are short these calls → short gamma → hedging buy-pressure exhausts near this strike → price stalls (ceiling / resistance).
Put 墙 — OTM put 行权价（现价下方 3–15%），得分最高。做市商 short 这些 put → 接近此位自动买入 → 支撑。

最佳区间：3–10% OTM。太近 = 磁吸（会被穿过）；太远 = 彩票 / 突破目标，非真实阻力。
强度：持续性 85%+ = 强（结构性），65–85% = 中等，50–65% = 弱.
Trade rules: Sell CC at call wall · Sell CSP at put wall · Don't buy lottos above call wall · Watch wall migration (up = bullish re-rating, down = top signal, thinning = resistance 弱ening).
⚠️ 不对称警告：密集 call 阶梯 + 稀薄 put 端 = 晚期牛市狂热（无对冲需求，市场自满）。
