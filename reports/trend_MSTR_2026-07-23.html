How GEX walls differ from volume walls (click)
Volume walls (below, Persistent Volume Strikes) show where retail/institutional flow concentrates — backward-looking conviction.
GEX walls (this card) show where dealer hedging pressure is largest — forward-looking, drives actual intraday price behavior.

A strike can be a volume wall without being a GEX wall (e.g. far-dated OI has low gamma). The GEX wall is what actually acts as resistance/support intraday because dealers must trade stock to hedge. When they agree, the wall is robust. When they disagree, trust GEX.

How to read OI Delta by depth (click to expand)
OI Delta = net new contracts created between snapshots (buyers + sellers both open). The headline total is misleading — where the OI lands matters more than the total.

OTM Call (above spot) = call OI increased above spot. Direction is unconfirmed without trade prints; read alongside price and volume.
ITM Call (below spot) = usually covered-call writers locking gains on existing stock. Increases OI but is not bullish — it's profit-taking.
OTM Put (below spot) = downside bets or portfolio hedges.
Deep OTM Put (<-15% spot) = systematic tail insurance. Only bought by institutions/macro funds. When this surges, smart money is nervous.

Classic patterns:
• Bullish candidate: OTM call OI expands while puts stay quiet, especially if price/volume confirm
• Bearish/hedge candidate: OTM put OI expands while calls stay quiet, especially if price/volume confirm
• "Seatbelt on": ITM calls + deep OTM puts both swell → institutions taking profit + hedging = late-bull warning
• Rotation: ITM calls closing (OI drops) + OTM calls opening (OI rises) → long positions being rolled up after a rally

How walls are identified (click to expand)
Call Wall — OTM call strike (3–15% above spot) with highest persistence × avg daily volume. Dealers are short these calls → short gamma → hedging buy-pressure exhausts near this strike → price stalls (ceiling / resistance).
Put Wall — OTM put strike (3–15% below spot) with highest score. Dealers short these puts → auto-buy on dips near strike → support.

Sweet spot: 3–10% OTM. Too close = magnet (gets crossed). Too far = lottery / breakout target, not active resistance.
Strength: 85%+ persistence = strong (structural), 65–85% = moderate, 50–65% = weak.
Trade rules: Use walls as levels first. Short-vol structures require Playbook and Rulebook approval. Watch wall migration (up = bullish re-rating, down = top risk, thinning = resistance fading).
⚠️ Asymmetry warning: Dense call ladder + thin put side = late-bull euphoria (no hedging demand, market complacent).
